---
name: using-ucas-mail
description: Instructions for using the UCAS mail system
---

To understand how to use the mail system, run the following command to get the latest instructions and protocols:

```bash
ucas mail instruction
```

This command will provide you with:
- Message formatting rules
- Available commands (`list`, `read`, `send`, `addressbook`)
- Your identity and mailbox location
- Protocols for communicating with USER and other agents

**Always follow the instructions provided by this command.**
