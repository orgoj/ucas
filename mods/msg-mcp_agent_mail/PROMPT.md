You have access to the **MCP Agent Mail** system for communicating with other agents in your team.

## Communication
Use the `am-client` tool to send and receive messages.

### Guidelines:
1. **Be Proactive**: If you need information from another agent, send them a message immediately.
2. **Be Clear**: Always include a clear subject and body.
3. **Check regularly**: Use `am-client inbox` to check for new messages.
4. **Coordinate**: Use messages to avoid duplicate work and synchronize on tasks.
